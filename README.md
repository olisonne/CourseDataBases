This program is designed to manage hotel data.

## Тема
13 Готель

Сучасне життя немислиме без ефективного управління інформацією. І для цього використовуються системи обробки інформації, засновані на базах даних (БД).
Від даних інформаційної системи багато в чому залежить ефективність роботи будь-якого підприємства чи установи.
Упорядкованість обліку наданих послуг дуже важлива для готелю. Узагальнення даних підвищує ефективність і спрощує роботу, до того ж база даних забезпечує зручний і швидкий доступ до інформації, підвищуючи швидкість роботи студії.
Мета: У процесі проектування необхідно розробити базу даних для предметної області «Готелі», яка автоматизує діяльність по обліку гостей, номерів і бронювань.

## Usage example
It will help you
- Manage customer information
	- Add new clients
	- Change clients
	- Delete clients
- Manage room information
	- Add new rooms
	- Change rooms
	- Delete rooma
- Manage service information
	- Add new servicrs
	- Change services
	- Delete services
- Booking and cancellation of rooms
